# Customer Behaviour Analysis – Excel Dashboard Project

![Dashboard Preview](Dashboard%20Image.png)

1️⃣ Project Background

This project analyzes customer transaction data to understand purchasing behavior, revenue distribution, and demographic contribution. The objective is to transform raw transactional data into actionable business insights using Microsoft Excel. This analysis simulates a real-world retail business scenario where management needs data-driven recommendations to improve revenue and customer strategy.

2️⃣ Objectives

- Identify primary revenue drivers
- Analyze customer demographics contribution
- Evaluate seasonal sales performance
- Understand shipping and payment preferences
- Generate strategic business recommendations

3️⃣ Data Description

Dataset Link: [Click here](https://www.opendatabay.com/data/consumer/ad9e2ab7-7559-4c89-af01-7d9df45b4255)

Dataset contains:
- 3,899 customers
- 1,075 subscribers
- Product categories (Clothing, Footwear, Accessories, Outerwear)
- Gender
- Age groups
- Seasonal transaction data
- Shipping types
- Payment methods
- Revenue and review rating

4️⃣ Data Processing Workflow

Step 1 – Raw Data Preparation
- Structured transaction-level dataset
- Checked for missing values
- Removed duplicates
- Standardized categorical values
- **Sheet: Raw Data**

Step 2 – Data Cleaning & Transformation
- Created calculated columns
- Categorized age into age groups
- Standardized revenue format
- Validated numeric consistency
- **Sheet: Clean Data**

Step 3 – Aggregation & KPI Calculation
- Using Pivot Tables to compute:
- Total Revenue
- Average Revenue
- Revenue by Category
- Revenue by Age Group
- Revenue by Season
- Shipping Usage
- Payment Method Distribution
- **Sheet: Pivot**

Step 4 – Data Visualization & Dashboard Design
- Designed interactive dashboard using Pivot Charts
- Implemented slicers (Gender, Age Group, Category, Season)
- Applied consistent green-themed visual layout
- Highlighted key KPIs at top level
- **Sheet: Chart & Dashboard**

Step 5 – Project Summary

Final executive summary sheet containing:
- Key KPIs
- Core findings
- Strategic recommendations
- **Sheet: Insight**

5️⃣ Key Findings

1. Clothing accounts for ~45% of total revenue, significantly higher than Accessories (~31%) and Footwear (~15%).
2. The 56+ age segment contributes ~26.8% of total revenue, indicating high purchasing power concentration in mature customers.
3. Seasonal revenue fluctuates within a ~6% range, with Spring as peak period and Summer as lowest.
4. Free Shipping usage exceeds other shipping methods by ~4–8%, suggesting delivery cost influences conversion behavior.
5. Revenue concentration is moderately distributed across age and payment segments, reducing dependency risk on a single customer profile.
