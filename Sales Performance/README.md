# Sales Performance Analysis – Excel Dashboard Project

![Dashboard Preview](Dashboard%20Image.png)

1️⃣ Project Background

This project analyzes retail sales performance to evaluate revenue trends, transaction volume, product contribution, and demographic distribution. The objective is to transform transactional sales data into measurable KPIs and business insights using Microsoft Excel. This project simulates a business scenario where management needs monthly performance monitoring and category-level revenue analysis.

2️⃣ Objectives

- Monitor overall sales performance
- Identify top-performing product categories
- Analyze monthly revenue trends
- Evaluate demographic contribution (gender & age group)
- Generate actionable business insights

3️⃣ Data Description

Dataset Link: [Click here](https://www.kaggle.com/datasets/mohammadtalib786/retail-sales-dataset)

Dataset contains:

- 1000 total transactions
- 1000 total customers
- Revenue data (monthly & categorical)
- Product categories (Beauty, Clothing, Electronics)
- Gender
- Age group

4️⃣ Data Processing Workflow

Step 1 – Raw Data Preparation
- Structured transactional dataset
- Checked for missing values
- Validated revenue & quantity fields
- **Sheet: Raw Data**

Step 2 – Data Cleaning & Transformation
- Standardized categorical values
- Created calculated revenue fields
- Validated transaction consistency
- **Sheet: Clean Data**

Step 3 – Aggregation & KPI Calculation
Using Pivot Tables to compute:
- Revenue per Month
- Revenue per Category
- Sales by Age Group
- Sales by Gender
- Total Transactions
- Average Revenue
- **Sheet: Pivot Table**

Step 4 – Data Visualization & Dashboard Development
- Built interactive dashboard using Pivot Charts
- Implemented slicers (Age Group, Gender, Product Category)
- Designed structured KPI layout
- Highlighted revenue & transaction metrics
- **Sheet: Charts & Dashboard**

Step 5 – Project Summary
Executive summary sheet containing:
- Key KPIs
- Core analytical findings
- Strategic implications
- **Sheet: Insight**

5️⃣ Key Findings

1. Electronics and Clothing generate the highest revenue (~$156K each), significantly outperforming Beauty (~$142K).
2. Monthly revenue fluctuates between ~$23K to ~$53K, with peak performance in May (~$53K).
3. The 50+ age group shows the highest purchase volume, indicating strong purchasing power concentration in older segments.
4. Sales distribution between male and female customers is relatively balanced, reducing demographic dependency risk.
5. Revenue concentration across categories remains diversified, lowering reliance on a single product line.
